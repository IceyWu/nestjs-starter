## Description

[Nest](https://github.com/nestjs/nest) framework stater with prisma redis alioss .


## start the app

```bash
$ pnpm install

# 生成迁移文件(数据库初始化)
$ npx prisma migrate dev

# development
$ pnpm dev

```

## 模块创建

```bash
$ nest g res article --no-spec
```
